# Morpion

# Introduction
Nous sommes en train de créer un jeu de Tic Tac Toe auquel vous pourrez jouer directement dans votre navigateur !

## Fonctionnalités
- Partie 2 joueurs sur le même appareil
- Interface simple et responsive
- Détection automatique de victoire, match nul et tours
- Bouton "Rejouer" pour relancer une partie instantanément.
